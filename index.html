About Me

Hola, nama saya Gamaliel atau sering dipanggil EL. Saya lahir di Biak, Papua. Saya sangat suka teknologi, terutama gadget. Saya bisa membuat proyek seperti hand landmark untuk mengendalikan perangkat dengan gerakan tangan, serta membuat game sederhana. Saya cukup jago dalam memodifikasi berbagai hal. Saya menikmati bermain game, menonton film, dan menjelajahi tempat baru. Cita-cita saya adalah mendapatkan pekerjaan dengan gaji 1 miliar/bulan.
